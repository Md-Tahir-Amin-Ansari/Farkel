About

Hello! My name is Md. Tahir Amin Ansari, and this project is a digital adaptation of the classic dice game, Farkle. I developed it as a way to practice my JavaScript skills and to better understand game development concepts.
The inspiration for this project came from a version of Farkle featured in the game Kingdom Come: Deliverance (KCD). I loved the strategic yet simple gameplay and wanted to recreate that experience while adding my own twist with custom scoring rules.
This project was built using HTML, CSS, and JavaScript, and I used WebStorm IDE for development. It includes features like:
  • A custom scoring system tailored for a fun and unique gameplay experience.
  • An interactive user interface designed to be intuitive and visually appealing.
